# SEO Guidelines — Shared

## On-Page SEO

### Title Tags
- Format: `Primary Keyword — Secondary Keyword | Brand`
- Max 60 characters
- Front-load the primary keyword

### Meta Descriptions
- 150-160 characters
- Include primary keyword naturally
- Include a call-to-action
- Mention data/numbers when possible

### Headings
- One H1 per page (matches title tag intent)
- H2s for main sections (include secondary keywords)
- H3s for subsections
- Don't skip levels (H1 → H3)

### Content Structure
- Answer the search intent in the first paragraph
- Use short paragraphs (2-3 sentences max)
- Include data tables where relevant (Google can extract featured snippets)
- Internal links in body text (not just CTAs)
- Image alt text should describe the data shown

## Blog SEO

### Article Structure
1. Hook: answer the query immediately (position 0 / featured snippet target)
2. Context: why this matters now (timeliness signal)
3. Data: tables, charts, specific numbers
4. Analysis: what the data shows (expert signal)
5. CTA: link to relevant tool/page on the site

### Word Count
- Data articles: 800-1,200 words
- Educational guides: 1,500-2,500 words
- Product updates: 400-600 words

### Publishing Cadence
- 4 articles/month minimum
- Mix: 2 data-driven + 1 educational + 1 product update
- Update existing articles when data changes significantly

## Link Building

### Internal Linking
- Every blog post links to 2-3 relevant site pages
- Network pages link to related blog posts
- Anchor text should be descriptive (not "click here")

### External Backlinks
- Target: 5-10 new backlinks/month
- Sources: Reddit, crypto forums, guest posts, data citations
- Approach: be helpful first, link second
- Never buy links or participate in link schemes

## Technical SEO (for content agent awareness)

- Site uses Next.js 14 with SSR (pages are server-rendered)
- Dynamic pages have structured data (JSON-LD)
- Sitemap auto-generated
- Content agent should not worry about technical SEO — focus on content quality

## Social SEO (Twitter/X)

- Tweets with links get less reach — put link in reply or thread
- Threads get more engagement than single tweets
- Engagement (replies) > reach (impressions)
- Best posting times for crypto: 9-11 AM UTC, 3-5 PM UTC
- Use 1-3 relevant hashtags, not walls of hashtags
